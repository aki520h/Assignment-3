# Travel Destinations Gallery - Assignment 3

This is a Full-Stack CRUD application built with *Node.js, Express, and MongoDB*. It allows users to manage a collection of travel destinations with image uploads.

## Features
- *Create:* Add new destinations with titles, countries, and photos.
- *Read:* View all saved destinations in a clean, structured table.
- *Update:* Edit existing destination details and update photos.
- *Delete:* Remove destinations from the database.
- *Security:* Sensitive credentials managed via .env file and protected by .gitignore.
- *Validation:* Clean and valid HTML5 structure (including Doctype, Head, and Body tags).

## Tech Stack
- *Backend:* Node.js, Express
- *Database:* MongoDB Atlas (Cloud)
- *ODM:* Mongoose
- *Templating:* EJS
- *File Uploads:* Multer
